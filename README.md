# Transcript Analyzer

A Python Flask application utilizing Python, Tailwind, and the OpenAI API to provide the user with an interface to transcribe data on their academic transcript.

The application will be hosted on AWS. 

### Technologies 

* Python
* Flask
* PyPDF
* Tailwind
* OpenAI API
* AWS

## Development Tools

* Visual Studio
* Git / Github
* AWS SDK
